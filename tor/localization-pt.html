Você pode usar este Rolador de Dados para jogar online. Se você abrir a janela do chat, rolagens que você fizer vão aparecer lá para os outros verem.
O Rolador de Dados tem 7 botões representando as diferentes quantidades de dados que você pode rolar no jogo:
  • Se você quer rolar apenas o dado de Proeza, clique no botão Proeza.
  • Se você quer rolar o dado de Proeza mais três dados de Sucesso, clique no botão ‘3’.
  • Você pode digitar um número na caixa de texto para fazer rolagens com um modificar dado. Por exemplo, para rolar o dado de Proeza com 2 dados de Sucesso e com um modificador +3 digite um 3 na caixa de texto, e então clique no botão ‘2’. Todas as rolagens serão feitas com o modificador até que você esvazie a caixa de texto.
  • Você pode fazer uma rolagem como se o personagem estivesse Exausto. Para isso, coloque uma marca na caixa Exausto. Assim, os dados de Sucesso com valores 1, 2 e 3 vão mostrar 0.
  • Você pode rolar o dado de Proeza duas vezes e manter o melhor ou pior resultado. Para isso, marque uma das duas caixas correspondentes. As rolagens serão feitas com o dado de Proeza rolado duas vezes e um dos resultados sendo descartado.

Bem-vindo ao Servidor Online de Personagens.
Aqui você pode criar grupos de personagens para compartilhar com seus amigos e outras pessoas. Os dados de personagem são armazenados no servidor gratuitamente.
Você pode usar estas características para jogar um jogo online. Com o chat você poderá compartilhar rolagens de dados.
Quando você salva um personagem no servidor, você e seus amigos vão poder abrir sua ficha de personagem e atualizá-la durante o jogo.
A primeira coisa que você precisa ter para poder usar todo o potencial do Servidor Online é criar seu próprio Grupo de Personagem.
Se você não tiver um (se esta for sua primeira vez aqui você provavelmente não tem) você precisará criar um.
Quando você criar um Grupo de Personagem você deve escolher um Nome Público do Grupo (que será mostrado para todos) e um login e senha privados.

Não escolha um login e senha que sejam fácil de adivinhar a partir do Nome Público do Grupo. Se outras pessoas adivinharem suas credenciais, seus personagens podem ser vandalizados!
